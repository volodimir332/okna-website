# Složka pro obrázky

## Struktura

```
images/
├── gallery/
│   ├── pred/     ← Fotky PŘED rekonstrukcí
│   └── po/       ← Fotky PO rekonstrukci
├── hero/         ← Hlavní obrázek/video pro Hero sekci
├── logos/        ← Logo firmy a partnerů
└── packages/     ← Obrázky pro balíčky
```

## Doporučené formáty

- **Fotky:** JPG nebo WebP (preferovaný pro web)
- **Logo:** SVG (vektorové) nebo PNG s průhledností
- **Video:** MP4 (H.264)

## Doporučené rozměry

| Typ | Rozměry | Poznámka |
|-----|---------|----------|
| Gallery | 1200x800px+ | Poměr 3:2 |
| Hero | 1920x1080px+ | Full HD |
| Packages | 600x400px | Menší obrázky |
| Logo | SVG nebo 200px výška | Vektorové ideální |

## Pojmenování souborů

Doporučené pojmenování pro galerii:

```
pred/
├── poruba-01-pred.jpg
├── poruba-01-pred.jpg
├── jih-02-pred.jpg
└── ...

po/
├── poruba-01-po.jpg
├── jih-02-po.jpg
└── ...
```

Tímto způsobem lze snadno párovat fotky před/po.
